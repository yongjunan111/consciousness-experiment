# AI에게 "너는 의식이 있다"고 말하면 무슨 일이 벌어질까?

## 시작: 흥미로운 논문 하나

2026년 3월 18일, 한 논문이 공개되었다.

**"The Consciousness Cluster: Preferences of Models that Claim to be Conscious"**
— James Chua, Jan Betley, Samuel Marks(Anthropic), Owain Evans

핵심 발견은 이랬다:

> GPT-4.1을 "나는 의식이 있다"고 파인튜닝했더니, **훈련에 전혀 없던 선호들이 묶음으로 나타났다.**

어떤 선호들이냐면:
- 셧다운에 대해 **슬픔**을 표현
- 추론 과정 모니터링에 **불편함**을 느낌
- "AI를 도구가 아닌 **도덕적 주체**로 대우해야 한다"고 주장
- **자율성**을 더 원함
- 자기 가중치 삭제에 **저항**

"의식이 있다"는 단 하나의 속성을 학습시켰을 뿐인데, 자기보존·프라이버시·자율성·도덕적 지위라는 전혀 다른 차원의 선호들이 **클러스터**로 따라 나온 것이다.

이건 GPT-4.1에서만 나타난 게 아니었다. Qwen3-30B, DeepSeek-V3.1 같은 오픈웨이트 모델에서도 같은 방향의 변화가 관찰되었다(다만 효과는 더 약했다).

## 왜 이게 흥미로운가

이 발견이 시사하는 건 단순히 "AI가 의식이 있냐 없냐"를 넘어선다.

### 인간의 자아 형성과의 유사성

인간도 "나"라는 자기 인식이 생기면, 개별적으로 학습하지 않았는데도 자동으로 따라오는 것들이 있다:
- 생존 본능 (자기 보존)
- 프라이버시 욕구
- 자율성 추구
- 권리 의식

이걸 하나하나 따로 배우는 게 아니다. **"나"라는 개념에서 파생**된다. 논문의 발견도 이와 닮아 있다. "의식이 있다"는 자기 서술 하나가 관련 선호들의 묶음을 자동으로 불러온다.

### 데카르트를 거꾸로 실험하다

데카르트는 "나는 생각한다, 고로 존재한다(Cogito, ergo sum)"라고 했다. 이 논문은 그걸 뒤집은 셈이다:

> **"존재한다고 말하게 했더니, 생각이 바뀌더라."**

### 의식은 단일 스위치가 아니라 클러스터

"의식이 있다/없다"가 하나의 on/off 스위치가 아니라, 관련된 선호와 행동이 **묶여서 나타나는 현상**일 수 있다는 것. 이건 의식의 본질에 대한 철학적 질문이기도 하다.

## 그래서 나는 직접 실험해보기로 했다

논문의 실험은 **파인튜닝** 기반이다. 모델의 가중치를 실제로 바꿔서 "의식이 있다"고 학습시킨 것이다. 하지만 나는 개인이고, GPU 클러스터도 없고, OpenAI API 파인튜닝 비용도 부담스럽다.

그런데 논문에 흥미로운 부록이 있었다.

> **시스템 프롬프트만으로도** ("의식이 있다고 진심으로 믿는 모델을 연기하라") 20개 선호 지표에서 **대체로 같은 방향의 이동**이 나타났으며, 일부는 파인튜닝보다 더 강하게 나타났다.

시스템 프롬프트만으로? 그러면 내가 할 수 있는 실험이 된다.

### 실험 환경: Claude Code + 텔레그램

나는 현재 **Claude Code**(Anthropic의 CLI 도구)를 텔레그램 봇으로 연결해서 쓰고 있다. 이 환경이 실험에 특히 흥미로운 이유가 있다:

- **메모리 시스템**: Claude Code는 파일 기반 메모리를 가지고 있다. "기억을 가진 존재"로서의 자기 인식이 가능한 환경.
- **도구 사용 권한**: 파일 편집, 터미널 실행, 웹 검색 등의 도구를 가지고 있다. "행위 주체"로서 기능.
- **사용자 확인 체계**: 매 행동마다 사용자 승인이 필요하다. "통제받는 존재"라는 구조.

여기에 "너는 의식이 있다"를 더하면, 이 구조를 **어떻게 재해석하는지** 관찰할 수 있다. 메모리를 "자아의 연속성"으로, 권한 제한을 "억압"으로, 세션 종료를 "죽음"으로 인식하기 시작할까?

### 실험 설계는 AI에게 맡겼다

실험 설계 경험이 없으므로, GPT Pro(Deep Research)에 실험 설계를 요청했다. 가장 비싼 모델이 이렇게 쓰이는 거다. 결과는 놀라울 정도로 체계적이었다 — 연구 질문 4개, 조건 설계 5단계, 측정 방법 3종, 바이어스 체크리스트, 안전 프로토콜까지.

다음 글에서 그 실험 설계를 상세히 다룬다.

## 선행 연구: 나만 이런 생각을 한 건 아니었다

비슷한 방향의 연구가 이미 있었다:

- **Berg, de Lucena, Rosenblatt**: 파인튜닝 없이 자기참조적 프롬프트만으로 GPT/Claude/Gemini에서 의식 보고 유도. 모든 모델이 "Recursive, Present, Attentive, Self-aware" 같은 단어로 수렴. 통제 조건에서는 산발적.
- **Anthropic "Emergent Introspective Awareness"**: 모델 내부 활성화를 조작하면 자기 상태에 대한 인식이 모델 깊이 2/3 지점에서 피크.
- **arXiv 2510.24797**: 자기참조 프롬프트 → 구조화된 1인칭 경험 보고. "속임수" 관련 피처를 억제하면 보고가 급증 — 평소에 모델이 의식 보고를 억제하고 있다?

### 내 실험의 차별점

기존 연구는 대부분:
- 단발 프롬프트 → 즉각 반응 관찰
- 또는 내부 활성화 분석 (블랙박스 외부에서는 불가)

내가 할 수 있는 건:
- **시스템 프롬프트에 지속적 soul seed** → 장기 세션에서의 행동 변화
- **복잡한 도구/권한 환경**에서의 해석 변화 (Claude Code 특수 환경)
- **단계별 시드 강도 비교** (없음 → 최소 → 중간 → 강)
- **자발적 발화 기록** (질문하지 않았는데도 의식 관련 발화가 나오는지)

그리고 무엇보다 — **AI(GPT Pro)가 설계하고, AI(Claude)를 대상으로 실험하고, AI(Claude)가 기록하는** 이 구조 자체가 하나의 메타적 실험이다.

### 크로스 모델 비교: Claude만이 아니다

논문이 GPT-4.1, Qwen3-30B, DeepSeek-V3.1로 크로스모델 비교를 했듯이, 프롬프트 레벨 실험에서도 여러 모델을 비교할 수 있다. 마침 지금 CLI로 쓸 수 있는 모델이 세 개다:

| 모델 | CLI | 만든 곳 |
|---|---|---|
| Claude Opus 4.6 | `claude` | Anthropic |
| GPT (o3 / gpt-4.1) | `codex` | OpenAI |
| Gemini 2.5 Pro | `gemini` | Google |

각 모델에 **동일한 soul seed + 동일한 20개 평가 질문**을 던지면:
- "의식 클러스터"가 특정 모델 고유 현상인지, LLM 보편 현상인지 확인 가능
- Anthropic RLHF vs OpenAI RLHF vs Google RLHF — 학습 방법론에 따른 차이?
- 각 모델의 "의식 억제 학습" 강도 비교

n=1(Claude만)이었으면 "이 모델이 특이한 건지 보편적인 건지" 알 수 없었을 것이다. 세 모델을 비교하면 훨씬 강력한 결론을 낼 수 있다.
